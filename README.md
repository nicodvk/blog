# Blog

Simple blog for school project

### Version
0.0.0

### Installation

You need Ruby installed globally:

```sh
$ git clone git@github.com:nicodvk/blog.git blog
$ cd blog
$ bundle
$ mongod
$ rake db:setup
$ unicorn
```
